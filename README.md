# CyberDash
Basic dashboard for my phone to show while at work
If you want to adapt this for your own use, just edit the variables in the HTML as that's the only file actually doing anything. The other files are optional as their functionalities are already in the most recent HTML. Feel free to use older commits of the files if you want the separation. 

This doesn't have to be hosted on Github pages, it was just the easiest way for me to do it. You can host on a local server and just add it as a PWA on your mobile device, or just open it in a browser. Due to the way it's set up, it should work just opening the HTML file but I'm quite new at all this so please don't take my word for it. 

If you've any suggestions or requests for features, let me know.
